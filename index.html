Web Development: Eggplantify App

MongoDB | NodeJS | Express | AngularJS | Materialize | Responsive CSS

How do you know if an image link is safe for work? Eggplantify is a Single-Page Application that uses the MEAN stack. Users can upload images, filter NSFW (not safe for work) images through an image recognition SDK and save SFW images to their gallery.

Full code on GitHub.

Web Development: Food Fight App

HTML | CSS | Materialize

Food Fight is an app for people who are undecided about what to eat. Users can sign in via Google through OAuth, input a food category and location and the app will display a random restaurant one at a time using the Yelp and Google Maps APIs. This was a group project of 4 people of which I was primarily the designer who chose the graphics and font and designed the UI.

Full code on GitHub.
